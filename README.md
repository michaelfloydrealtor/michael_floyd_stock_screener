Michael Floyd's Stock Screener

Interactive stock analysis web application built with Python and Streamlit.

Live App:

https://michaelfloyd-stock-screener.streamlit.app

Features!

Interactive stock data visualization

Fetches live market data via yFinance

Calculates:

SMA 50 (Simple Moving Average)

RSI (Relative Strength Index)

Clean charts built with Matplotlib

User-selectable ticker input

Deployed on Streamlit Cloud

Lightweight, fast, and easy to use

 Technology Stack

Python 3

Streamlit

Pandas

NumPy

Matplotlib

yFinance

Installation (Local)
git clone https://github.com/YOUR_USERNAME/michael_floyd_stock_screener
cd michael_floyd_stock_screener
pip install -r requirements.txt
streamlit run app.py

 Indicators
SMA 50

Smooths price data to identify long-term trends.

RSI (14)

Measures price momentum and signals overbought/oversold levels.
